Dstatypes in JavaScript

  • String

    A collection of characters.
    Example : "John"

  • Number

    Any number (decimal, whole, negative numbers).
    Example: 3.14

  • Boolean

    True or False values.
    Example: true (Note: JavaScript is case-sensitive)

  • Undefined

    A variable which is declares but not assigned with any value.
    var x

  • Function

    Any function declared with keyword 'function'.
    Example: function myFunc(){}

  • Object

    An instance of a class.
    Example: {name:'John', age:34}

The keyword 'typeof' can be used to datatype.
Example: typeof 3.14         (Returns 'number')
