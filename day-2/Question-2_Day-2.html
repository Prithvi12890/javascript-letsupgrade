String and Array methods

String methods

  • search(content)

    The search(content) method searches a string for a specified value and returns the position of the first match.

    Example:

         var str = "Please locate where 'locate' occurs!";
         var pos = str.search("locate");
         console.log(pos);



  • slice(start,end)

    slice() extracts a part of a string and returns the extracted part in a new string.
    The method takes 2 parameters: the start position, and the end position (end not included).

    Example:

          var str = "Apple, Banana, Kiwi";
          var res = str.slice(7, 13);
         console.log(res);



  • substring(start,end)

    substring() is similar to slice().
    The difference is that substring() cannot accept negative indexes.

    Example:

         var str = "Apple, Banana, Kiwi";
         var res = str.substring(7, 13);
         console.log(res);



  • substr(start,lenght)

    substr() is similar to slice().
    The difference is that the second parameter specifies the length of the extracted part.

    Example:

         var str = "Apple, Banana, Kiwi";
         var res = str.substr(7, 6);
         console.log(res)



  • replace(word,it's substitute)

    The replace() method replaces a specified value with another value in a string (only the first value).

    Example:

         str = "Please visit Microsoft!";
         var n = str.replace("Microsoft", "Google");
         console.log(n);



  • trim()

    The trim() method removes whitespace from both sides of a string

    Example:

         ' Hello World! ';
         alert(str.trim());



  • charCodeAt(position)

    The charCodeAt() method returns the unicode of the character at a specified index in a string


    Example:

         var str = "HELLO WORLD";
         console.log(str.charCodeAt(0));



Array methods

  • toString()

    The JavaScript method toString() converts an array to a string of (comma separated) array values.

    Example:

         var fruits = ["Banana", "Orange", "Apple", "Mango"];
         console.log(fruits.toString());



  • delete

    Since JavaScript arrays are objects, elements can be deleted by using the JavaScript operator delete.
    Using delete may leave undefined holes in the array. Use pop() or shift() instead.

    Example:

         var fruits = ["Banana", "Orange", "Apple", "Mango"];
         delete fruits[0];
         console.log(fruits);



  • splice()

    The splice() method can be used to add new items to an array.
    The first parameter defines the position where new elements should be added (spliced in). The second parameter defines how many elements should be removed.
    The rest of the parameters define the new elements to be added.

    Example:

         var fruits = ["Banana", "Orange", "Apple", "Mango"];
         fruits.splice(2, 0, "Lemon", "Kiwi");
         console.log(fruits);



  • slice()

    The slice() method slices out a piece of an array into a new array.

    Example:

         var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
         var citrus = fruits.slice(1);
         console.log(citrus);
